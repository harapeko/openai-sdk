## Installation

- [Installation Hatch](https://hatch.pypa.io/latest/install/)
- Set Environment Variable `OPENAI_API_KEY`
  - Get API KEY https://openai.com/

## Run

- `cd path/to/repository`
- `hatch run python main.py`
